# Third-party notices

This first commit contains no imported application source, prompt text, models, fonts, or runtime dependencies beyond files generated by the repository itself. This document establishes the required disclosure record before upstream material is brought in.

## Required handling

When third-party material is added, the same change must:

1. preserve the upstream copyright and license notice;
2. record the upstream URL and immutable commit/version in `UPSTREAMS.md`;
3. identify each carried file as verbatim, adapted, new, or excluded;
4. inventory the selected subset's direct and transitive dependencies; and
5. include the corresponding source/build instructions required for distributed binaries.
